Inconvenientes con un pago

Esta gestión se utiliza cuando un cliente nos informa que realizó un pago pero este no imputó en sistema o cuando el cliente abona por error en otra cuenta.
Se podrá cargar un Inconveniente con Pago cuando el cliente posea el comprobante en su poder.
Veamos el paso a paso del proceso de Inconvenientes con un Pago.

¡Atención!

En este caso la gestión quedará derivada a Cobranzas que se encargará de verificar que el comprobante sea válido y que el pago no esté imputado en esa ni en otra cuenta.
Si el pago fue realizado en Rapipago completá el lugar de pago, la dirección del Rapipago y el Cajero (si se tiene el dato) para que Cobranzas pueda reclamarlo.


Cuando un cliente se contacta con el canal telefónico por un inconveniente con un pago se deberá seguir el siguiente paso a paso:

  1. Representante del canal telefónico toma la gestión y le indica al cliente que deberá enviar el comprobante a documentacion@teco.com.ar.
  2. El Back Office toma el caso que quedó derivado y adjunta el comprobante que llegó a la casilla de mail.
  3. El Back Office toma el caso que quedó derivado y adjunta el comprobante que llegó a la casilla de mail.

Promesa de pago

Si el cliente no posee el comprobante a mano, dentro de la gestión Inconvenientes con un Pago se podrá cargar una Promesa de Pago.

¡Atención!

Al cargar la Promesa de Pago la línea se habilitará por 7 días. El cliente deberá comprometerse a enviar el comprobante dentro de este período de tiempo.

Encontrá info en BC con las siguientes palabras clave: Inconveniente, Pagos, Deuda, Inconveniente Pago, Pago, Inconveniente Promesa De Pago, Factura, Inconveniente Con Pago, Inconvenientes.

Informe de Pago

La imputación real de un pago en Salesforce será de forma inmediata.

¿Qué beneficios implica esto?

  • Podrás ver el pago reflejado en el Resumen de Cuenta del cliente en el momento.
  • La habilitación del servicio será automática si este se encontrara suspendido por mora.

A modo de contingencia para esta salida, se habilitará la funcionalidad de Informe de Pago a través de IVR para aquellas entidades que no tengan disponible el Cobro Online. Recordá que desde el CRM podremos cargar, tal como vimos previamente, una Promesa de Pago.
